📌 Project Overview

AutoDeployX — Automated DevSecOps CI/CD Platform

AutoDeployX is an end-to-end DevSecOps project where developer code travels from GitHub to production servers automatically — without manual steps (only after one manual approval).

When a developer executes git push, a GitHub Webhook notifies Jenkins. Jenkins then runs an automated pipeline — code is built, Docker images are created, security is scanned, pushed to DockerHub, and finally deployed to an AWS EC2 server.

This project was created as a college final year project following real-world industry practices.
