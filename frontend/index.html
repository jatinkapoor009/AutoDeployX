📌 Project Overview

AutoDeployX — Automated DevSecOps CI/CD Platform

AutoDeployX ek end-to-end DevSecOps project hai jisme developer ka code GitHub se automatically production server tak pahunchta hai — bina kisi manual step ke (sirf ek approval ke baad).

Jab developer git push karta hai, GitHub Webhook Jenkins ko notify karta hai. Jenkins automated pipeline chalata hai — code build hota hai, Docker images banti hain, security scan hoti hai, DockerHub pe push hota hai, aur finally AWS EC2 server pe deploy ho jaata hai.

Yeh project college final year project ke roop mein banaya gaya hai jo real-world industry practices follow karta hai.
